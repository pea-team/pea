# peajs

> A React Framework based on TypeScript.

## Quick Start

```bash
npm i -g pea-cli
pea new myapp
cd myapp
npm run dev
```

It will create a directory called myapp inside the current folder:

```bash
.
├── package.json
├── src
│   └── App.tsx
└── tsconfig.json
```

Then open：http://localhost:3000

## Scripts

- `npm run dev`
- `npm test`
- `npm build`
