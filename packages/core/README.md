# @peajsi/core

## Installation

```sh
npm i @peajs/core
```

## Api

### `bootstrap`

```js
import Pea from '@peajs/core'

Pea.bootstrap()
```

### `updateConfig`

```js
import Pea from '@peajs/core'

Pea.updateConfig({ foo: 'bar' })
```
